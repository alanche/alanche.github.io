RubyTips

  1. Query the instance methods defined a in specific class Foo?
    • Foo.instance_methods(false) : false is instruct to ignore the inherited methods.
    • you can use the Array#grep methods to further select the result: e.g. String.instance_methods.grep /^re/
  2. list the class Array methods?
    • [].methods.grep /^re/
  3. add additional methods to a specific class ?
    • add additional method to class String : 
         class String
     def to_aphanumeric
       gsub(/[^\w\s]/,'')
     end
   end
      site effects: the added methods may overrides the existing method.
  4. Module.contants Return all the top level constants. 
    Module.contants.include? :Object # True
Module.contants.include? :Module # True
  5. Module#contants return the contants defined in current scope 
    module M
  Y = "A contant"
  class C
  end
end
M.constaints # [:C,:Y]
  6. instance methods and class methods are different 
    String.instance_methods == "abc".methods # True
String.methods == "abc".methods # False
  7. class is also a object 
    "hello".class # String
String.class  # Class
Array.superclass # Object
Object.superclass # BasicObject
BasicObject.superclass # nil
Class.superclass # Module
